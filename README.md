# EXPRESS TYPESCRIPT STARTER TEMPLATE

## Installation

Use the pnpm to install foobar.

```bash
pnpm install
```

## Usage

Running on local environment

```bash
pnpm run dev
```

Build

```bash
pnpm run build
```
